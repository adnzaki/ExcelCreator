Implementation Playground

1. Basic Data Export Writer.php

use ExcelTools\Writer;

$excel = new Writer();


$data = [

['Product Name', 'Stock', 'Price'],

['Mechanical Keyboard', 45, 89.99],

['Wireless Mouse', 120, 24.50]

];


$excel->fillCell($data, 'A1');

$excel->save('Inventory_Report.xlsx');

2. Advanced Formatting & Merging Writer.php

use ExcelTools\Writer;

$excel = new Writer();


// Configure dimensions & fonts

$excel->setDefaultFont('Arial', 12);

$excel->mergeCells('A1:C1');

$excel->fillCell([['MONTHLY FINANCIAL REPORT']], 'A1');


// Apply rich array styles directly

$headerStyle = [

'font' => ['bold' => true, 'color' => ['rgb' => 'FFFFFF']],

'fill' => ['fillType' => $excel->fill::FILL_SOLID, 'color' => ['argb' => 'D9E1F2']],

'alignment' => ['horizontal' => $excel->alignment::HORIZONTAL_CENTER]

];

$excel->applyStyle($headerStyle, 'A1:C1');


// Adjust layouts dynamically

$excel->setMultipleColumnsWidth(['A', 'B', 'C'], 20);

$excel->setRowHeight(1, 35);

$excel->save('Styled_Report.xlsx');

3. Multi-Sheet Orchestration Writer.php

use ExcelTools\Writer;

$excel = new Writer();


// Sheet 1 (Default Sheet)

$excel->spreadsheet->getActiveSheet()->setTitle('Active Users');

$excel->fillCell([['John Doe', 'Admin']], 'A1');


// Create and switch to Sheet 2 cleanly

$newSheet = $excel->spreadsheet->createSheet();

$newSheet->setTitle('Archived Logs');

$excel->setActiveSheet(1); // Switch by index pointer


$excel->fillCell([['Old Log Entry', '2024-12-10']], 'A1');

$excel->save('MultiSheet_System.xlsx');

4. Advanced File Reading & Associative Array Parsing Reader.php

$reader->loadFromFile($_FILES['excel_file']['tmp_name']);


// Fetch data using the first row as array keys ('withHeaderRow' = true)

foreach ($records as $row) {

API Method Reference

Class: ExcelTools\Writer Methods

save(string $filename, string $excelType = 'Xlsx') void

Saves the spreadsheet file and forces an immediate browser stream download to the client, preconfigured with the appropriate Content-Type headers automatically.

applyStyle(array $style, string $range) void

Applies mass style configurations based on an array scheme (borders, fonts, alignments, fills) to a specifically targeted range of cell coordinates.

fillCell(array $value, string $startCell = 'A1') void

Hydrates multi-dimensional matrix data loops into the worksheet directly, starting from the specified anchor cell coordinate.

wrapText(string $cell) void

Activates automatic wrap text functionality on a specific cell so that long text segments do not spill over adjacent boundaries horizontally.

mergeCells(string $cells) / unmergeCells(string $cells) void

Merges or unmerges a block array group of designated coordinate excel cells (e.g., 'A1:C1').

setColumnWidth(string $dim, $width = null) / setDefaultColumnWidth(int $width) void

Configures column dimensions. If the width argument is passed as `null`, the system immediately enables responsive AutoSize tracking to match content text lengths.

setMultipleColumnsWidth(array $dimensions, $width = null) void

Efficiently alters widths for several distinct columns simultaneously within a single line map statement (e.g., `['A', 'B', 'D']`).

setRowHeight($dim, $height) / setMultipleRowsHeight($dims, $height = null) void

Directs custom row height constraints. Supports expressive range formats (e.g., strings like `'1-5'` or variable nested arrays).

setDefaultFont(string $font, int $size) void

Defines a global typography typeface dynamic and fallback font scale across the entire active worksheet workbook before cellular modifications take place.

Class: ExcelTools\Reader Methods

loadFromFile(string $filepath) void

Auto-identifies spreadsheet types, processes raw incoming binaries from local target server paths, and loads data directly into active memory objects.

getSheetData(bool $withHeaderRow = false) array

Extracts all data row entities into native arrays. When set to `true`, the first row maps as associative column keys, while underlying Excel `DateTime` format tokens are cleanly formatted into regular date string values (`Y-m-d`).

getSheetNames() array

Fetches an index directory list of all available worksheets stored inside the currently loaded workbook package.

setActiveSheet($identifier) void

Shifts runtime read/write context pointer focusing on a specific worksheet sheet index (`int`) or explicit workbook name identifier (`string`).
